# 🖩 MARIE Scientific Calculator

A scientific calculator built in Python with Tkinter, styled around the MARIE 
(Machine Architecture that is Really Intuitive and Easy) CPU model — 
operations are tracked through simulated AC (Accumulator), PC (Program Counter), 
and IR (Instruction Register) values, updated live after every calculation.

## Tech Stack
- Python
- Tkinter (GUI)

## Features
- Standard operations: add, subtract, multiply, divide, power
- Scientific functions: square root, sin, cos, tan, log, ln, factorial, absolute value
- Live register display (AC, PC, IR) that updates after each operation
- Input validation and error handling (e.g. divide by zero, negative sqrt/log)
- Clear/reset functionality
